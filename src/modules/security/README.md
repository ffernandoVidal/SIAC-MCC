# Módulo: Seguridad y Acceso

Estructura inicial para los componentes de seguridad:

- gestión de usuarios
- roles y permisos
- asignación de roles
- validación de acceso (middleware / guardas)
- bitácora básica

## Archivos sugeridos

- `users.js` / `users.ts`
- `roles.js` / `roles.ts`
- `permissions.js` / `permissions.ts`
- `auth.js` / `auth.ts` (login, tokens, sesiones)
- `audit.js` / `audit.ts` (bitácora)
