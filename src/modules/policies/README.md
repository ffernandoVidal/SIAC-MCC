# Módulo: Pólizas (Operación contable)

Estructura inicial para el registro y validación de pólizas:

- encabezado de póliza
- detalle de póliza
- numeración y estados (borrador, aprobada, anulada)
- validaciones de cuadratura, períodos cerrados, cuentas inactivas
